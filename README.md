# Landing Page - Plant Future

This landing page belongs to The Odin Project from the Flexbox chapter.

## Built with
 
- HTML
- CSS

## Structure
 
The page consists of four main sections and a footer:
 
1. **Hero**: header with title, text, and image.
2. **Info**: cards with images and descriptions.
3. **Quote**: highlighted section with a featured quote.
4. **Call to action**: call-to-action block.
5. **Footer**: footer block.
